Function for main comunication with water level sensor @detail Function read all water-sensors and compute adecvate watter-level. Function also checks for degradation of sensor(in case of corosion). In case that sensors are not flooded sequtialy, function returns value 255 as ERROR state. For reading without checking error state use force_read_level();.
